# weightzz

Objetivo:
Permitir registrar progressão de carga na academia.

Usuário:
Praticantes de musculação.

Problema:
Hoje usam papel, notas ou planilhas, ou nem mesmo registram sua progressão de carga, tornando seus treinos menos eficientes.

Funcionalidades:

- Criar treinos
- Criar exercícios
- Registrar séries
- Registrar peso
- Registrar repetições
- Visualizar progressão por periodo de tempo

Não faz:

- Rede social
- Chat
- Marketplace
