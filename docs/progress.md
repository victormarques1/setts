# Progressão de Carga

## Objetivo

Permitir que o usuário acompanhe a evolução da carga utilizada em cada exercício ao longo do tempo.

## Fonte dos Dados

A progressão é calculada a partir dos registros históricos de SetRecord.

## Métrica Principal

Para cada sessão de treino:

Selecionar a maior carga utilizada no exercício.

Exemplo:

01/06
Supino
70kg x 10

05/06
Supino
75kg x 8

10/06
Supino
80kg x 6

Resultado:

01/06 → 70kg
05/06 → 75kg
10/06 → 80kg

## MVP

A tela deve exibir:

- Exercício selecionado
- Histórico cronológico
- Gráfico simples de evolução

## Fora do escopo

- Volume total
- 1RM estimado
- Estatísticas avançadas
- Comparação entre exercícios
- IA
- Recomendações
